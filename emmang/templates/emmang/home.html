App

Emmang (Employee Mannagment) es un sistema de administración básico, que ayuda con la comunicación de un equipo de trabajo. Actualmente solo es un prototipo que funciona para 1 solo equipo de trabajo, pero escalarlo para el manejo de varias empresas/equipos no representa una mayor complicación.

Herramientas de desarrollo

Se utilizó Semantic UI para agilizar el proceso de Frontend Development. SASS se encarga de los detalles. Django funciona como Backend para el proyecto.
